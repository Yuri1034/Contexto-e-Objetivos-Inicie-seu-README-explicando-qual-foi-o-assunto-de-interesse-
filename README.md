🧾 Resumos Estruturados
🔐 Criptografia

A criptografia é uma técnica utilizada para proteger informações, transformando dados legíveis em um formato codificado. Apenas quem possui a chave correta pode acessar o conteúdo original.

Simétrica: usa a mesma chave para criptografar e descriptografar.
Assimétrica: utiliza um par de chaves (pública e privada).
Aplicação prática: proteção de arquivos, comunicação segura e armazenamento de dados sensíveis.
🦠 Ameaças Cibernéticas

Ameaças digitais são técnicas utilizadas para roubar, danificar ou acessar dados sem autorização.

Ransomware: bloqueia dados e exige pagamento.
Keylogger: registra tudo o que o usuário digita.
Phishing: engana o usuário para roubar informações.
Essas ameaças exploram falhas técnicas e, principalmente, o comportamento humano.
🛡️ Mecanismos de Defesa

A segurança digital depende de múltiplas camadas de proteção:

Antivírus: detecta e remove ameaças.
Firewall: controla o tráfego de rede.
Sandboxing: executa arquivos em ambiente isolado.
Conscientização do usuário: evita ataques baseados em engenharia social.

Boas práticas incluem manter sistemas atualizados, usar senhas fortes e evitar downloads suspeitos.

📚 Glossário de Conceitos
Criptografia: processo de codificação de dados para proteção.
Descriptografia: processo inverso, retornando ao formato original.
Chave criptográfica: elemento usado para codificar/decodificar dados.
Malware: software malicioso.
Ransomware: tipo de malware que sequestra dados.
Keylogger: ferramenta que captura teclas digitadas.
Firewall: sistema de controle de tráfego de rede.
Phishing: técnica de fraude para roubo de dados.
Sandbox: ambiente isolado para testes seguros.
Engenharia social: manipulação psicológica para obter informações.
